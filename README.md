# OrderInChallenge

Web project to solve Taco challenge.

## Table of Content

- [Run](#run)
- [Test](#test)

## Run

```bash
# Go into the folder with solution and run:
docker-compose up
```
Service will be available on port 8888.

When docker runs open http://localhost:8888.

## Test
```bash
# Go into the folder with solution and run:
dotnet test OrderInChallenge.sln
```
